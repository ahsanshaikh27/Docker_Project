🚀 Ubuntu Apache Web Server – Dockerized
This project demonstrates how to containerize a simple Apache web server using Ubuntu and Docker. It is a basic project that helps beginners understand Dockerfile usage, image creation, container execution, and serving static web pages.

📚 Topics Covered
This project covers the following key Docker concepts:

✅ Using a base image (ubuntu:latest)

✅ Setting environment variables (ENV)

✅ Updating packages and installing Apache (apt-get)

✅ Cleaning up unnecessary files (apt-get clean)

✅ Exposing ports (EXPOSE)

✅ Defining working directory (WORKDIR)

✅ Copying files into the image (COPY)

✅ Running Apache in the foreground (CMD)
